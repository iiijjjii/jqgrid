How to install/upgrade a new jquery-ui
======================================

1. Download jquery ui package and unzip its content here.
   (Actually only its css/* and js/* are needed.)

2. Make adjustment::

    cp js/jquery-ui-1.x.x.custom.min.js js/jquery-ui.custom.min.js
    cp css/smoothness/jquery-ui-1.x.x.custom.css css/smoothness/jquery-ui.custom.css

3. Done! (Note that client browser might need a cache reload to reflect the change.)
